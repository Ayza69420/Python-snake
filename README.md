# Pygame snake

Snake game made in python with the pygame library.

# Requirements

- pip
- pygame

# Pygame Installation

On the command line, type:

```pip install pygame```

# How to Install

Installation with git

``git clone https://github.com/Ayza69420/Python-snake``   

# Running the program

``cd Python-snake/``   

Windows

``python main.pyw``  

UNIX Systems

```python3 main.pyw```
